Getting Started

1. This program contains 3 main modules (panels). You can access each one simply by clicking on the tab.

2. All the table elements can be sorted by alphabetical order or reverse alphabetical order.  Simply click on the header of the desired column

   3. Pull down the dropdown menu and select the type you want to search. The search box support sub strings, this means you can enter part of the name you want to find.

4. These data files are critical to the functionality of this program.

    Deleting any of them will cause malfunction

5. The program saves your progress after every operation. Feel free to exit at anytime.

    Simply click on the "close" button located on the top right hand corner.

Admin Panel

Add Item/Patron [back to top]

Step 1. Click the add button

Step 2. Fill in all the fields with the correct information

              The program will displays an error message if any part is missing or wrong.

Step 3. Click "Save" to add the Item/Patron into the data base.

            You should see the new entry appear in the list located on the right.

Modify Item/Patron  [back to top]

Step 1. Select the Patron/Item from the list on the right

Step 2. Click the modify button

Step 3. Change the desired field(s) NOTE: You are not allowed to decrease the "amount out" for "Item"

Step 4. Click "Save" to modify the entry

            When you select the Item/Patron, the information should be changed.

Delete Item/Patron  [back to top]

Step 1. Select the Patron/Item from the list on the right

Step 2. Click the delete button

Step 3. Verify your action by clicking "yes" if you want to delete the entry, or "no" if the delete button was accidently clicked.

Borrow and Return Panel

 

The patron's borrow and return information can be view by entering the PatronID

The patron's borrowed item should be displayed on the tab below. If the due date is in red, that means it is overdue.

This program also keeps permanent record of borrowed history for each patron. It can be accessed by clicking on the "Borrow History" tab.

 

Borrowing  [back to top]

Step 1. Enter the 6-digit PatronID

Step 2. Select a book to borrow

Step 3. Set today's date (default: program will retrieve system date)

Step 4. Click "Borrow"

            You should see the book appearing in the "Borrowed" tab.

Returning  [back to top]

Step 1. Enter the 6-digit PatronID

Step 2. Select a book to return from "Borrowed" tab

Step 3. Click "Return"

            You should see the book disappear from the "Borrowed" tab. It can still be found under "Borrowed History" tab.

Report Compilation Panel

Lending Item Report [back to top]

All the items in the library can be compiled and printed out in order of title or type. Simply click the corresponding buttons below.

Checked out Report [back to top]

All the checked out items in the library can be compiled and printed in order of the date it is due, type, and the patron who borrowed it.

Printing [back to top]
